# Sistema Avanzado de Medición de Reflejos

## Descripción

Sistema basado en ESP32 que mide el tiempo de reacción de uno o dos jugadores ante estímulos aleatorios visuales o sonoros.

## Hardware

- ESP32
- 3 LEDs
- 1 Buzzer activo
- 8 Pulsadores (4 por jugador)
- 1 Pulsador de inicio
- 1 Pulsador modo Simón
- 1 Pulsador finalizar juego
